# Brew setup
#### This script installs all necessary requirements for web development

## Requirements
#### Mac Os
* macOS Mojave (10.14)
* macOS Catalina (10.15)
* macOS BigSur (11.0 - 11.4)

## Install
Download the script
```
curl --remote-name https://raw.githubusercontent.com/leo-midili/brew/main/brewinstall.sh
```
Execute the script
```
./brewinstall.sh
```
